#### Current weather API
```http
  GET https://api.openweathermap.org/data/2.5/weather?q={city name}&appid={API key}
```
#### Forecast  API
```http
  GET https://api.openweathermap.org/data/2.5/forecast?lat={lat}&lon={lon}&appid={api key }&units=metric`;
```
